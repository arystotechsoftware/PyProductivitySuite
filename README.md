# ArystoTech Free Apps

`ArystoTech Free Apps` is ArystoTech's lineup of free Windows utility apps.

The bundle binary is named `PySuperApp.exe`, and the free lineup includes both the bundle app and individual standalone downloads.

It bundles six focused tools into one PySide6 desktop app so users can handle file cleanup, image workflows, JSON cleanup, and log extraction from a single install.

## Included Tools

### PyBulkRenamer

- Preview original and renamed filenames side by side before applying changes.
- Combine prefix, suffix, and find/replace rules in one pass.
- Best for cleaning up folders and standardizing naming conventions safely.

### PyImageSquisher

- Compress one image or batch-process a drag-and-drop queue.
- Reuse export format, max width, and quality settings across single and bulk workflows.
- Reports progress and estimated space savings during batch runs.

### PyImageConverter

- Convert a single image or a full drag-and-drop batch queue.
- Detects supported input and output formats at runtime from Qt image plugins.
- Supports SVG rendering, icon canvas sizing, matte color handling, and smaller-file optimization.
- Automatically keeps output filenames unique during batch exports.

### PyJson2CsvConverter

- Convert a JSON file or pasted JSON text into CSV.
- Flattens nested objects and helps surface record arrays for export.
- Best for turning API payloads and exports into spreadsheet-ready rows.

### PyJsonXmlFormatter

- Format, minify, and validate JSON or XML quickly.
- Useful for debugging, documentation, and cleaning up payloads.

### PyLogExtractor

- Search large logs using plain text or regular expressions.
- Processes line by line and writes matches to a separate output file.
- Best for isolating errors, IDs, IPs, and recurring patterns from big log files.
